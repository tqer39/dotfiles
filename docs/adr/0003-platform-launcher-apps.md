# ADR-0003: 各プラットフォームのランチャーアプリ選定

## ステータス

Accepted

## コンテキスト

dotfiles のセットアップにおいて、各プラットフォームで統一的なユーザー体験を提供するため、キーボードランチャーアプリを導入する。ランチャーアプリは、アプリケーションの起動、ファイル検索、計算、クリップボード管理などの機能を提供し、生産性を向上させる。

### 要件

- キーボードショートカットで即座に起動できること
- アプリケーション検索・起動ができること
- 拡張機能やプラグインをサポートしていること
- 各プラットフォームで安定して動作すること
- 自動インストールが可能であること

## 検討した選択肢

### macOS

#### 選択肢 1: Raycast

- モダンな UI とパフォーマンス
- 豊富な拡張機能 (Store)
- クリップボード履歴、スニペット、ウィンドウ管理を統合
- 無料版で十分な機能
- Homebrew でインストール可能

#### 選択肢 2: Alfred

- 長年の実績と安定性
- Powerpack (有料) で拡張機能が利用可能
- Homebrew でインストール可能

#### 選択肢 3: Spotlight (標準)

- 追加インストール不要
- 拡張性が限定的

### Windows

#### 選択肢 1: Raycast (Windows 版)

- macOS 版と同じ操作感
- 2024年リリースのベータ版
- winget でインストール可能

#### 選択肢 2: PowerToys Run

- Microsoft 公式ツール
- 無料でオープンソース
- プラグインサポートあり

#### 選択肢 3: Keypirinha

- 軽量で高速
- プラグインで拡張可能

### Linux (Ubuntu)

#### 選択肢 1: Albert

- Qt ベースで高速
- 豊富なプラグイン
- apt でインストール可能 (OpenSUSE リポジトリ経由)

#### 選択肢 2: Ulauncher

- Python ベースで拡張しやすい
- モダンな UI

#### 選択肢 3: Rofi

- 軽量で高速
- X11/Wayland 両対応
- 設定ファイルベース

## 決定

以下のランチャーアプリを採用する:

- **macOS**: Raycast
- **Windows**: Raycast
- **Linux**: Albert

## 理由

### macOS: Raycast

| 観点 | Raycast | Alfred | Spotlight |
| ---- | ------- | ------ | --------- |
| 機能の豊富さ | ◎ | ◎ (有料) | △ |
| 拡張性 | ◎ | ◎ | × |
| 無料利用 | ◎ | △ | ◎ |
| UI/UX | ◎ | ○ | ○ |
| 自動インストール | ◎ | ◎ | - |

### Windows: Raycast

| 観点 | Raycast | PowerToys Run | Keypirinha |
| ---- | ------- | ------------- | ---------- |
| macOS との統一感 | ◎ | × | × |
| 機能の豊富さ | ○ | ○ | ○ |
| 自動インストール | ◎ | ◎ | △ |
| 開発の活発さ | ◎ | ◎ | △ |

### Linux: Albert

| 観点 | Albert | Ulauncher | Rofi |
| ---- | ------ | --------- | ---- |
| パフォーマンス | ◎ | ○ | ◎ |
| 拡張性 | ◎ | ◎ | ○ |
| UI の洗練度 | ◎ | ◎ | △ |
| 自動インストール | ◎ | ○ | ◎ |
| ドキュメント | ○ | ◎ | ○ |

### 主な決定理由

1. **クロスプラットフォームの統一性**: macOS と Windows で Raycast を使用することで、操作感を統一できる
2. **機能の豊富さ**: Raycast は無料版でもクリップボード履歴、スニペット、計算機などの機能を提供
3. **自動インストール対応**: すべての選択肢が各プラットフォームのパッケージマネージャーでインストール可能
4. **アクティブな開発**: Raycast と Albert はともに活発に開発されており、長期的なサポートが期待できる

## 結果

以下の方法でインストールを自動化:

- **macOS**: `config/packages/Brewfile` に `cask "raycast"` を追加
- **Windows**: `install.ps1` の winget パッケージに `Raycast.Raycast` を追加
- **Linux**: `scripts/installers/apt.sh` に `install_albert()` 関数を追加
